# Manual Blog Import Guide

Since the programmatic migration is encountering issues, here's the manual approach to import your blog data:

## Step 1: Access Admin Interface
1. Start the CMS: `pnpm dev`
2. Go to `http://localhost:3000/admin`
3. Create admin user if not done already
4. Navigate to "Blog Posts" collection
